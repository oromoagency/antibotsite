NexAPI Cloud Reference

Welcome to the official NexAPI Cloud documentation. Our RESTful API allows you to programmatically manage your infrastructure, retrieve analytics, and configure your API gateways.

Authentication

All API requests require authentication using a Bearer token. You can generate an API key from your dashboard.

# Example request with authentication
curl -H "Authorization: Bearer nx_live_7x89ab..." \
     https://api.nexapi-cloud.com/v1/users

Endpoints

Retrieve a list of users in your organization.

GET /api/demo/v1/users 
// Response (200 OK)
{
  "data": [
    {
      "id": "usr_1",
      "name": "John Doe",
      "role": "admin"
    }
  ],
  "meta": {
    "total": 1
  }
}

Fetch real-time metrics for your API gateway.

GET /api/demo/v1/metrics

Create a new API key (requires admin privileges).

POST /api/demo/v1/keys 
// Request body
{
  "name": "Production Key",
  "scopes": ["read", "write"]
}
